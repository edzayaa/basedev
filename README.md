# BaseDev MDX

Codebase to guide your development process.

- CSS media queries structure.
- HTML boilerplate.
- Project structure.

## Run Locally

1. Clone the project

```bash
  git clone git@github.com:edzayaa/basedev.git
```

2. Go to the project directory

```bash
  cd basedev
```

3. Start the server with Live Server
